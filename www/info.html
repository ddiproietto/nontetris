Nontetris - Info

This game is a rewriting of Not Tetris 2 by Maurice Guégan.

It is far from complete, I hope to add more features and fix glitches soon

The game is entirely written in C++ (not a single line of javascript has been hand-written) and compiled in javascript using cheerp. The same codebase compiles on PC and on the web.

Source code is at http://github.com/ddiproietto/nontetris/

The project uses Box2D and WebGL.

  • Box2D was succesfully ported to compile and run with cheerp by me and Leaning Technologies. This port will be released as open source soon.
  • WebGL has been used through a small wrapper for the GLES API. Like the rest of the code, it has been written exclusively in C++.
